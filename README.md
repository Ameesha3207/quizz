Quizz App
Project Overview

The Quiz App is a simple, interactive web application built using HTML, CSS, and JavaScript. It allows users to answer multiple-choice questions and tracks their score. The app is colorful, responsive, and beginner-friendly, making it perfect for learning web development basics.
Features
Multiple-choice quiz questions
Instant feedback (Correct / Wrong)
Next question navigation
Displays total score at the end
Colorful and interactive user interface
Easy to customize questions
Technologies Used
HTML – for the structure of the app
CSS – for styling and colors
JavaScript – for functionality, scoring, and interactivity
How to Run
Clone or download the repository to your local machine.
Ensure you have the following files in the same folder:
index.html
style.css
script.js
Open index.html in a web browser (Chrome, Firefox, Edge, etc.).
The first question will load automatically.
Click on an option to select your answer.
Click Next to go to the next question.
At the end, your total score will be displayed.
